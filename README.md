# Portfolio 

Portfolio is a custom WordPress theme that uses the popular HTML Canvas Theme as seen on Theme Forest website. 

## WordPress installation

In order to install WordPress you can go to the link below and simply click on Get WordPress and extract to your web root.
https://github.com/s7web/dockerWP

## Usage

In order to activate this theme, go to Dashboard and navigate to Appearance -> Themes -> Add new, and simply place your
downloaded .zip file, or extract it in folder wp-content/themes then activate it.
